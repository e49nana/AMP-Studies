## 📖 About

Complete coursework archive from my **B.Sc. in Applied Mathematics & Physics (B-AMP)** at **TH Nürnberg** — organized semester by semester.

This repo contains notebooks, cheatsheets, exercises, and project work from each module, serving both as a study reference and a record of my academic journey toward **MSc @ TUM** + **Exchange @ École Polytechnique (EuroTech)**.
